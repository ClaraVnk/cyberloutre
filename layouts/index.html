À propos

Médecin qui code, parce que pourquoi pas.

Je suis médecin en France, et quand je ne soigne pas, je code. Surtout pour mes confrères et consœurs — parce qu'on a besoin d'outils qui marchent et qui ne nous coûtent pas un rein.

Mes terrains de jeu : sites pour soignants, home automation (Home Assistant 4 ever), petits outils Python qui résolvent des frictions du quotidien. Tout est open source ici.

Je travaille à la cool freelance — pas d'agence, pas de mensualité gonflée, pas de "stratégie digitale globale 360". Juste un site qui marche, qui est joli, et qui coûte ce qu'il coûte vraiment.

Sur mesure

Outil clinique web

Aide-mémoire, calculateur, protocole interactif, base de ressources… Un outil qui te fait gagner du temps en consultation ou en formation. Co-construit, validé par tes confrères, accessible, RGPD-native.

Exemple : transidenticlic.com

Intégration

CMS + newsletter pour tes billets

Interface d'édition simple type WordPress sans les soucis de maintenance, et envoi automatique de ta newsletter chaque semaine via un service gratuit (jusqu'à 10k abonnés). Tu publies depuis l'admin, le reste se fait tout seul.

Stack typique : Decap CMS + Kit + Hugo + GitHub.

Aussi WordPress

Sites & maintenance WordPress

WordPress reste une option solide pour beaucoup d'usages (sites institutionnels, e-commerce léger, accès admin pour plusieurs contributeurs). Création, refonte, maintenance, sécurité, performance — quand c'est le bon choix, je le fais aussi.

Thèmes custom, plugins, sauvegardes, mises à jour.

Conseil

Audit / regard extérieur

Tu as déjà un site et tu veux un avis honnête ? Je regarde performance, accessibilité, RGPD, UX, sécurité, SEO. Tu repars avec un rapport actionnable, sans baratin commercial.

Format : 1h de visio + doc synthèse PDF.
